# Developer Portfolio Backend

REST API for a personal developer portfolio.

The backend manages projects, skills, contact messages, and administrator authentication.

## Features

* JWT Authentication
* Admin Login
* Protected Routes
* CRUD Operations
* Projects API
* Skills API
* Contact Messages API
* MongoDB Atlas Integration
* Image Upload Support
* Express REST API

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* Multer
* bcryptjs
* CORS
* dotenv
